PAMFLET PRODUCT


  • Pamflet 1

    • Diketahui :

    x = Jam Dinding

    y = Tas

    z = Jam Hias

    Model :

    1. x + y + z = 120.000

    2. 2y + z = 130.000

    3. 2x + y + z = 150.000

  • Eliminasi persamaan 1 & 2

    • x + y +z = 120.000 = x2

    2y + z = 130.000 = - = x1

    2x + z = 110.000 (pers 4)

  • Eliminasi persamaan 3 & 1

    • 2x + y + z = 150.000 = x2

    x + y + z = 120.000 = x1

    2x + y= -20.000 (pers 5)

  • Eliminasi pers 4 & 5

    • 2x + z = 110.000

    2x + z = 60.000 -

    z = 50.000

  • Subtitusi z = 50.000 ke pers 4

    • 2x + z = 110.000

    2x + 50.000 = 110.000

    2x = 110.000 - 50.000

    x = 60.000 : 2

    x = 30.000

  • Subtitusi x = 30.000 , z = 50.000 ke persamaan 1

    • x + y + z = 120.000

    30.000 + y + 50.000 = 120.000

    y + 80.000 = 120.000

    y = 120.000 - 80.000

    = 40.000
