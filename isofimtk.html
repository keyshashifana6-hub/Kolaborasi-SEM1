PAMFLET PRODUCT


  • Panflet 3

    • Diketahui :

    x = Jam

    y = Lampu

    z = Tas

    Model :

    1. x + y + z = 115.000

    2. 2x + y + z = 145.000

    3. x + y + 2z = 155.000

  • Eliminasi persamaan 2 & 1

    • x + y + z = 115.000

    2x + y + z = 145.000 -

    -x = -30.000

    x = 30.000 (pers 4)

  • Eliminasi persamaan 3 & 1

    • x + y + z = 155.000

    x + y + z = 115.000 -

    z = 40.000 (pers 5)

  • Subtitusi x = 30.000 dan z = 40.000 ke persamaan 1

    • x + y + z = 115.000

    30 + y + 40 = 115.000

    y + 70 = 115.000

    y = 115.000 -

    70.000

    = 45.000

    HP {= 30.000, 40.000, 45.000 }
