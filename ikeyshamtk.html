PAMFLET PRODUCT


  • Panflet 2

    • Diketahui :

    x = DVD

    y = Rak

    z = Jam

    Modal :

    1. x + y + z = 150.000

    2. 2x + y + z = 215.000

    3. x + 2y + z = 205.000

  • Eliminasi persamaan 2 & 1

    • 2x + y +z = 215.000

    x + y + z = 150.000 -

    x = 65.000 (pers 4)

  • Eliminasi persamaan 3 & 1

    • x + 2y + z = 205.000

    x + y + z = 150.000 -

    y = 55.000 (pers 5)

  • Subtitusi x = 65.000 dan y = 55.000ke pers 1

    • x + y + z = 150.000

    65.000 + 55.000 + z = 150.000

    120.000 + z = 150.000 - 120.0000

    z = 30.000

    HP = {65.000, 55.000, 30.000 }
